# Модуль :superkassa-server

## Назначение
Серверная часть ККМ Superkassa, предоставляющая HTTP REST API для внешних клиентов.

## Содержимое
- Spring Boot REST-контроллеры для программирования ККМ, проведения чеков, кассовых операций и получения диагностической информации.
- Настройка Spring Beans и привязка портов ядра к JDBC/J2SE-адаптерам.
- Конфигурации OpenAPI/Swagger.

## Архитектурные ограничения
- Java 21, Spring Boot 3.5.x.
- Управляет версиями библиотек через платформу Spring Boot BOM.
